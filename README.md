# FFFU

FFFU (FUSE Flickr For U) is a FUSE filesystem that allows the upload of generic files to Flickr as encoded `.png`s.

FFFU aims to adapt Flickr to be used as a general-purpose file system

----

Requirements:
-------------
    fuse    
    python 2.7.3
    beautifulsoup4
    flickrapi 1.4.5
    fs
    requests
